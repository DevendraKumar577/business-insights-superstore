# Business Insights & KPI Dashboard

## Project Overview
This project focuses on analyzing business performance data and presenting key insights through a KPI-driven dashboard.  
The goal is to understand overall sales and profit trends and summarize performance across categories, regions, and years using clear and structured analysis.

## Key Analysis
- Analyzed overall business performance using total sales and total profit KPIs.
- Studied sales distribution across product categories to identify top-performing categories.
- Analyzed profit trends over different years to observe growth patterns.
- Evaluated region-wise profit performance to compare business contribution across regions.

## Tools & Technologies
- Power BI – for creating interactive dashboards and visual analysis  
- Python – for data processing and summary generation  
- Jupyter Notebook – for analysis workflow  
- CSV files – for structured data storage  
- GitHub – for version control and project sharing  

## Files in Repository
- business_insights_project.ipynb – data analysis and summary generation  
- Sample_Superstore.csv – raw dataset used for analysis  
- category_summary.csv – category-level sales summary  
- region_summary.csv – region-wise profit summary  
- yearly_summary.csv – year-wise sales and profit summary  
- README.md – project documentation  

## Insights
This project demonstrates how business data can be transformed into meaningful insights using KPIs and dashboards.  
The analysis helps in understanding performance trends, identifying strong business areas, and supporting data-driven decision-making.
